* 설치 -- [Jekyll]을 먼저 설치한다. 
```
$ cd psj890729.github.io
$ bundle install
```
* 로컬에서 실행하기 
```
$ bundle exec jekyll serve
$ open http://localhost:4000
```
* 배포 -- github에 push만 해주면 됨

[Jekyll]: https://jekyllrb.com/
